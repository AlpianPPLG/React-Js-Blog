## Simple Blog App

Simple Blog App adalah aplikasi web sederhana yang memungkinkan pengguna untuk membuat, mengedit, dan menghapus posting blog. Aplikasi ini dibangun menggunakan React JS dan Tailwind CSS untuk tampilan dan manajemen UI, serta menggunakan React Router untuk navigasi antar halaman.

## Fitur Utama

- Tambahkan Posting Baru: Pengguna dapat membuat posting baru dengan mengisi judul, konten, dan nama penulis.

- Edit dan Hapus Posting: Pengguna dapat mengedit atau menghapus posting yang sudah ada.

- Pencarian: Fitur pencarian memungkinkan pengguna untuk mencari posting berdasarkan judul, penulis, atau konten.

- Responsif: Desain responsif untuk pengalaman pengguna yang baik di berbagai perangkat.

## Teknologi Yang Di Gunakan

- React JS
- Tailwind CSS
- React Router
- Framer Motion (untuk animasi)


## Instalasi

1. Clone Repository

        git clone https://github.com/username/simple-blog-app.git

2. Install Dependencies

        cd simple-blog-app

        npm install

3. Menjalankan Aplikasi

        npm start

        Aplikasi akan berjalan di http://localhost:3000.
